### _val_  and  _var_

```java
// Java
final String name = "Joe Smith";
```

```csharp
// C#
readonly string name = "Joe Smith";
```

```kotlin
// Kotlin
val name: String = "Joe Smith"
```

<br />

```java
// Java
int age = 23;
```

```csharp
// C#
int age = 23;
```

```kotlin
// Kotlin
var age: Int = 23
```

Note:
+ val simple: Int? // has type Int, default getter, must be initialized in constructor
